# User Interface

## System Requirements
### Install Dependencies
Install **Docker Desktop** from https://docs.docker.com/desktop/setup/install/windows-install/<br><br>
**Download Node Version Manager** from https://github.com/coreybutler/nvm-windows/releases <br>
**node:** version 25<br>
``nvm install 25.2.1``<br>
``nvm use 25.2.1``<br><br>
**npm:** version 11.6.2 \
``npm install 11.6.2``<br>

### Libraries <br>
**Material UI:** https://mui.com/material-ui/all-components/.<br>
**FontAwesome:** https://www.npmjs.com/package/@fortawesome/free-solid-svg-icons.<br>

## Running the UI
In the console run ``npm start`` to run the UI locally. The UI will run on http://localhost:3000.
